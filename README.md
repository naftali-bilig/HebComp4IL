# JClock for Galaxy Watch

פרויקט Wear OS המשמר את מימוש ה־JClock המקורי בתוך WebView מקומי. מודול `wear` הוא יישום השעון; מודול `mobile` הוא היישום הנלווה לטלפון ומספק מיקום רק בעקבות בקשה מהשעון.

„מיקום קבוע” מבקש מהטלפון נקודת GPS אחת ושומר אותה לחישובי השעון. „מיקום נייד” מרענן את המיקום בכל 6 שניות; חזרה ל„ירושלים” מפסיקה את הרענון. מיקום קבוע ונייד משתמשים באזור הזמן המקומי של הטלפון כפי שהוא, ורק מצב ירושלים משתמש ב־`Asia/Jerusalem`.

אם אין טלפון מחובר, אין הרשאה בטלפון או שאין מיקום זמין — השעון אינו משתמש ב־GPS מקומי ואינו מחליף את המיקום בערך שמור.

## מבנה המאגר

- `mobile` — אפליקציית הטלפון, כולל מעטפת WebView מאובטחת ל־`jclock.net`.
- `wear` — אפליקציית Wear OS.
- `zepp` — אפליקציית Zepp/Amazfit.
- `jclock-web` — קוד אתר JClock המלא, כולל `BirthCalculator`.
- `site` — רכיבי האתר הנוספים של הפרויקט.
- `output/latest` — התוצרים הבנויים האחרונים לבדיקה.

מטמוני בנייה, ערכות SDK/JDK מקומיות, `node_modules`, היסטוריות Git מקוננות
ומפתחות חתימה פרטיים אינם חלק מקוד המקור שהועבר למאגר זה.
