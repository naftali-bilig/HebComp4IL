# יום-טיוב

ספריית הסרטונים החיצונית נמצאת באותו R2 של ספריית המנגינות, תחת
`TuningLimud/`.

הקובץ `TuningLimud/manifest.json` צריך להיראות כך:

```json
{
  "files": [
    {
      "name": "mahasrot.mp4",
      "title": "סרטון הסבר על מעשרות"
    }
  ]
}
```

שמות הקבצים חייבים להסתיים ב-`.mp4` ואינם יכולים לכלול תיקיות משנה.
יש לאפשר בקונפיגורציית CORS של R2 בקשות `GET` מהדומיין של האפליקציה.

עד שהספרייה החיצונית זמינה, הנגן מציג את קובץ ה-MP4 המקומי שבתיקיית
`media/`.
